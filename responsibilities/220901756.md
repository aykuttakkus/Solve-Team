#### 2. ROC Curve and AUC Analysis (Bonus — Beyond Week 4)

To provide threshold-agnostic evaluation, I implemented ROC (Receiver Operating Characteristic) curve analysis and computed AUC (Area Under the Curve) for all models.

**Why ROC/AUC matters here:** All models reported above achieve similar point-estimate metrics (Accuracy, F1). ROC analysis evaluates performance across *all possible classification thresholds*, not just the default 0.5 cutoff. This is especially important in the medical context where the operating threshold may be adjusted to prioritize Recall (catch more Malignant cases) at the cost of Precision.

**Findings:** All models — Logistic Regression, L2-regularized MLP, and Early Stopping MLP — produced AUC ≈ 0.999. This confirms that the dataset is geometrically simple enough that the linear model captures nearly all discriminative information. The small performance gap between Plain MLP and regularized models observed in the classification metrics corresponds to threshold-level instability, not a difference in the underlying discriminative power of the learned representations.

#### 4. Academic Conclusions

**The Complexity Trap:** The Plain MLP demonstrated that adding architectural capacity without constraint degrades generalization on small datasets. The 30 features include pairs with correlations exceeding 0.99 (identified in Step 1). An unconstrained optimizer assigns large, unstable weights to these redundant features — a behavior that L2 regularization directly suppresses.

**Regularization Parity:** Both L2 and Early Stopping restored performance to the Logistic Regression baseline. This finding has a precise theoretical interpretation: on a dataset with high geometric simplicity (AUC ≈ 0.999), the performance limit is set by the data's intrinsic separability, not the model's representational power. Regularization's role is to prevent the model from *underperforming* this limit, not to exceed it.

**Is a more complex model necessary?** No — and this is a valid and important scientific conclusion. The Logistic Regression model achieves near-optimal performance because the data is nearly linearly separable in its 30-dimensional feature space. The neural network experiments provide value not by exceeding this performance, but by empirically demonstrating *why* complexity without constraint fails, and *how* regularization restores the theoretical optimum.
